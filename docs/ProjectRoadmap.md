# Project Roadmap

## Phase 1 - Foundation

* Configure Raspberry Pi management workstation.
* Install AWS CLI.
* Install Terraform.
* Install Ansible.
* Configure Git and GitHub.

## Phase 2 - AWS Infrastructure

* Create VPC.
* Create subnets.
* Create security groups.
* Deploy EC2 instance.
* Configure IAM.

## Phase 3 - Platform Services

* Deploy Docker.
* Deploy Nginx Proxy Manager.
* Deploy Authelia.
* Deploy Grafana.
* Deploy Prometheus.
* Deploy Loki.

## Phase 4 - Automation

* Configure GitHub Actions.
* Automate Terraform validation.
* Automate deployments.

## Phase 5 - Hybrid Connectivity

* Deploy Tailscale or WireGuard.
* Connect BirdNET-Go Raspberry Pi.
* Centralise monitoring and logging.

## Phase 6 - Kubernetes

* Deploy k3s.
* Migrate selected services.
* Implement GitOps workflows.
